BCIPRO: A non-invasive Brain-Computer Interface for a 3D-printed Prosthetic simulation

Nora Fadul, Seyed Mohammad Ghaemi, Nicholas Gresh, Zachary Gurwitz, Wensen Liu, Daniel Lopez, Nikhil Mittu, Abhay Patel, Khadija Shah, Micaela Wolcott

Advised by Dr. Anil Deane

Gemstone Honors Program

Spring 2020 - Spring 2023

keywords: Neuroscience, machine learning, prosthetics, VR/AR, 3D-printing, HCI

Collaborated as part of an interdisciplinary team of undergraduate students to test and train existing neural networks, processing acquired EEG signals from OpenBCI headset into mechanical operations for a 3D-printed prosthetic. I developed a 3D simulation environment in Unity with realistic physics-based interactions for an operational upper-arm and hand prosthetic to be used in rehabilitation. I created a multimodal, customizable, and accessible immersive VR interaction system for the Oculus Quest 2 and HTC Vive using SteamVR, OpenVR, and Unity XR Interaction Toolkit, integrating with our brain-computer interface.

Rendering Large Point Cloud Data for Virtual Reality in Unity using LOD Techniques

Daniel Lopez, Rami Shehadeh

Project led by Daniel Brown, advised by Dr. Roger Eastman

Colloration between NASA Mixed Reality Engineering Toolkit (MRET) and UMD Immersive Media Design (IMD) Lab

Fall 2022 - Spring 2023

keywords: graphics, rendering, point clouds, optimization

Explored point cloud rendering methods, including octree structures for level-of-detail (LOD) reduction, frustrum and occlusion culling, and point decimation. We utilized multiple open-source renderers to develop an end-to-end system capable of streaming large point cloud data (100 million points) in Unity while ensuring real-time performance (60 fps) for VR display.

Decluttering the Stars: A Study in AR Space Interface Minimalism

Jason-Alexander Fotso-Puepi, Damian Figueroa, Daniel Lopez, Alexander Wang, Vikaas Venkatesh, Oliver Adkins, Nhi Tran, Amy Tran, Kevin Gu

Advised by Dr. Matthias Zwicker

Selected Finalists of NASA Spacesuit User Interface Technologies for Students (NASA SUITS) Competition

Fall 2022 - Spring 2023

keywords: VR/AR, UI/UX, Design, Navigation

Member of the UMD TerpVISIO team that proposed and designed a comprehensive UX for an AR user interface, intended for use by Artemis astronauts during a simulated lunar test mission. Developed and implemented a feature-rich UI in Unity for the Microsoft Hololens 2, leveraging the Microsoft Mixed Reality Toolkit (MRTK), including visual and interactive elements for navigation such as customizable waypoints and pathfinding. Invited to conduct testing and iterative refinement of our system during on-site trials as Selected Finalists for the NASA student competition at the Johnson Space Center in Houston.

Persistent Distractor-based Redirection for Natural Walking in Small Physical Environments

Niall Williams, Daniel Lopez

keywords: HCI, VR/AR, Navigation

Collaborated with PhD Candidate Niall Williams and advised by Dr. Aniket Bera to develop a 3D procedurally generated Unity scene that demonstrates the practicality of persistent distractors in virtual environments for VR applications. Spot-It-Garbage

Fall 2019
Android Studio, Google Maps API, Firebase, Java, XML

Spot-It Garbage is an Android app built by me and a group of peers which allows users to pin a location with litter/waste on a Google Maps display and submit details about the site like pictures, its challenge for removal, and a link to a group chat to organize a cleanup. Other users can then view these pins and click on them to see details and join the cleanup via the group chat. The app is based off the principle of crowd-sourcing and social media, allowing users to easily define areas in need of waste management and to recruit help from volunteers.

Spot-It Garbage is a native Android app written in Java and XML, with a Realtime Database and Cloud Storage using Firebase. Individual users are able to pin garbage locations and through using Firebase's Realtime Database, these markers are publicly visible. We relied on the Google Maps API to provide the front end experience (displaying the map, finding location, placing markers, etc.). App was developed as a part of HackUMBC 2019.
